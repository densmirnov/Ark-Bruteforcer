# Ark-Bruteforcer

Bruteforce your missing Ark passphrase and Ledger Nano S seed words.


### Installation and Usage
1) Install git
2) Clone this repository
3) ```cd Ark-Bruteforcer```
4) ```npm install```
5) ```node bruteforcer.js```
6) Follow the on-screen instructions
